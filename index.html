Para que serve o atributo "placeholder"?

O atributo placeholder é uma pequena frase ou palavra, que tem o intuito de ajudar usuários a entender como devem preencher determinado formulário, inserindo um texto contextual no campo a ser respondido

Para que serve o atributo "checked"?

O checked atributo é um atributo booleano. Quando presente, ele especifica que um elemento deve ser pré-selecionada (marcada), quando a página é carregada. O checked atributo pode ser usado com e .

Para que serve o atributo "value"?

Nos inputs text o value é o que a pessoa digita, nos checks e radios são nomes que nós colocamos para fazer sentido quando recebermos essa informação na outra ponta.

Na tag form
para que serve o atributo "action"?

O atributo action define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados. O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST" (veja as diferenças aqui).

Na tag form
para que serve o atributo "method"?

Para a tag podem ser atribuídos o atributo method e o atributo action. O atributo action define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário. O atributo method define o método HTTP com que o formulário HTML irá lidar com os dados recebidos.
